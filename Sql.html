SQL

SQL (Structured Query Language) is the standard programming language used to interact with and manage data stored in relational databases. It enables users to create, read, update, and delete data efficiently, making it essential for data management, data analysis, and application development.

What Is SQL?

  • SQL is designed to store, query, and manipulate data in relational databases, which organize data into tables of rows and columns.
  • Widely supported by database systems such as MySQL, PostgreSQL, Oracle, SQL Server, and SQLite.
Command Purpose
SELECT Retrieve data from one or more tables
INSERT Add new rows/records to a table
UPDATE Modify existing records in a table
DELETE Remove specified records from a table
CREATE Build new databases, tables, or objects
DROP Delete databases, tables, or objects
