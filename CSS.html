CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML or XML. It controls how web pages look, including layout, colors, fonts, and animations. Think of it this way:

  • HTML is the skeleton and content of the webpage (headings, paragraphs, images).
  • CSS is the clothing and style (making those headings red, arranging paragraphs in columns, adding background images).

Key Points

  • Cascadingmeans styles from different sources (external, internal, inline) follow priority rules; more specific styles and those marked !important take precedence
  • CSS allows for responsive web design, adapting layouts and styles to various screens (desktop, tablet, mobile).
  • CSS modules (Color, Layout, Animations, etc.) are standardized by the W3C and evolve individually for new features.
